## 1. 快速开始

### 1.1 什么是微连？

微连是一款桌面应用，让你可以在微信中远程操控电脑上的 AI 编程助手（如 Claude Code、OpenCode 等）。

### 1.2 核心功能

- **多 Agent 支持**：切换使用不同的 AI 工具
- **微信遥控**：在微信中发送消息，桌面 AI 回复
- **会话管理**：多会话支持，消息历史保存
- **数据同步**：WebDAV 云同步，多设备数据一致
- **安全保护**：API Key 本地加密存储

### 1.3 适用场景

- 在手机上查看 AI 的代码建议
- 远程操作家里的电脑进行代码审查
- 多台电脑之间快速切换 AI 工具

## 2. 安装指南

### 2.1 系统要求

| 系统 | 版本要求 |
|------|----------|
| Windows | Windows 10 或更高 |
| macOS | macOS 12 (Monterey) 或更高 |
| Linux | Ubuntu 20.04 LTS 或更高 |

### 2.2 下载安装

#### Windows

1. 下载 `微连-Setup-0.1.0-win.exe`
2. 双击运行安装程序
3. 按提示完成安装
4. 桌面会出现「微连」图标

#### macOS

1. 下载 `微连-0.1.0.dmg`
2. 双击挂载 DMG 文件
3. 将「微连」拖拽到「应用程序」文件夹
4. 在启动台找到并打开应用

#### Linux

1. 下载 `微连-0.1.0.AppImage`
2. 添加执行权限：`chmod +x 微连-0.1.0.AppImage`
3. 双击运行：`./微连-0.1.0.AppImage`

### 2.3 首次启动

1. 双击打开应用
2. 等待初始化（约 2-3 秒）
3. 应用启动后，会自动扫描系统中已安装的 AI 工具

## 4. Agent 管理

### 4.1 什么是 Agent？

Agent 是安装在电脑上的 AI 编程助手 CLI 工具，例如：

- **Claude Code**：Anthropic 的 AI 编程助手
- **OpenCode**：OpenAI 的代码助手
- **CodeBuddy**：通用代码生成工具

### 4.2 查看 Agent 列表

1. 点击导航栏「🤖 助手」
2. 列表显示所有已发现的 Agent
3. 每个卡片显示：
   - Agent 名称
   - 所属厂商
   - 状态（在线/离线）
   - 是否为默认 Agent

### 4.3 切换 Agent

**方法一：在 Agent 列表切换**

1. 进入「助手」页面
2. 点击想要使用的 Agent 卡片
3. 卡片高亮表示已选中

**方法二：在微信中切换**

1. 发送消息：`/agent claude`
2. 机器人回复：「已切换到 claude」

### 4.4 安装新 Agent

#### 方法一：应用内一键安装

1. 进入「助手」页面
2. 点击「安装 Agent」按钮
3. 选择要安装的 Agent
4. 复制安装命令
5. 在终端中执行命令

#### 方法二：手动安装

**安装 Claude Code**

```bash
# macOS / Linux
npm install -g @anthropic-ai/claude-code

# Windows
npm install -g @anthropic-ai/claude-code

# 登录
claude auth login
```

**安装 OpenCode**

```bash
# macOS
brew install openai/openai-code/openai-code

# Windows
winget install OpenAI.OpenAICode
```

### 4.5 配置 Agent

1. 进入「助手」页面
2. 点击 Agent 卡片的「⚙️」按钮
3. 可配置项：
   - **工作目录**：AI 默认操作目录
   - **模型**：选择使用的模型
   - **参数**：温度、最大 Token 等

### 4.6 重新扫描

如果新安装了 Agent 但未显示：

1. 进入「助手」页面
2. 点击右上角「重新扫描」按钮
3. 等待扫描完成（约 2 秒）

## 5. 微信集成

### 5.1 微信扫码登录

1. 进入「设置」页面
2. 点击「微信连接」选项
3. 显示二维码
4. 用微信扫描二维码
5. 在微信中确认登录

### 5.2 在微信中发送消息

**发送普通消息**

```
Hello, world!
```

**使用斜杠命令**

```
/agent claude      # 切换到 Claude Agent
/clear             # 清空当前会话
/help              # 显示帮助信息
```

**发送文件（暂不支持）**

目前微信端仅支持文本消息，文件上传功能在开发中。

### 5.3 查看微信连接状态

- **已连接**：状态栏显示绿色图标
- **未连接**：状态栏显示红色图标，点击可重新连接

### 5.4 断开微信连接

1. 进入「设置」页面
2. 点击「微信连接」选项
3. 点击「断开连接」按钮

## 6. 会话管理

### 6.1 创建新会话

1. 进入「💬 聊天」页面
2. 点击列表顶部的「＋ 新会话」
3. 新会话自动切换为当前会话

### 6.2 切换会话

1. 进入「💬 聊天」页面
2. 点击左侧列表中的会话
3. 内容区域显示该会话的消息历史

### 6.3 重命名会话

1. 进入「💬 聊天」页面
2. 鼠标悬停在会话标题上
3. 点击「✏️」图标
4. 输入新名称并保存

### 6.4 删除会话

1. 进入「💬 聊天」页面
2. 鼠标悬停在会话上
3. 点击「🗑️」图标
4. 确认删除

⚠️ **注意**：删除会话会永久删除该会话的所有消息，无法恢复。

### 6.5 清空会话消息

1. 进入「💬 聊天」页面
2. 点击右上角「清空消息」按钮
3. 确认清空

### 6.6 导出会话

1. 进入「💬 聊天」页面
2. 点击右上角「导出」按钮
3. 选择导出范围：
   - **当前会话**：仅导出当前会话
   - **Agent 所有会话**：导出该 Agent 的所有会话
4. 选择保存位置
5. 导出为 Markdown 文件

## 7. 配置说明

### 7.1 工作目录

**作用**：AI Agent 默认操作文件的目录

**设置方法**：

1. 进入「设置」页面
2. 点击「工作目录」选项
3. 选择目录（Windows/macOS）或输入路径（Linux）

**默认值**：
- Windows: `C:\Users\{用户名}\Documents`
- macOS: `/Users/{用户名}/Documents`
- Linux: `/home/{用户名}`

### 7.2 系统提示词

**作用**：为所有 Agent 设置统一的系统提示词

**设置方法**：

1. 进入「设置」页面
2. 点击「系统提示词」选项
3. 输入提示词内容
4. 点击保存

**示例**：

```
你是一个专业的编程助手，请用中文回答问题。
```

### 7.3 密码保护

**作用**：保护 API Key 等敏感信息

**首次设置**：

1. 进入「设置」页面
2. 点击「密码保护」选项
3. 输入主密码（至少 8 位）
4. 勾选「信任此设备」（可选）
5. 点击保存

**解锁密码**：

1. 进入「设置」页面
2. 点击「密码保护」选项
3. 输入主密码
4. 点击解锁

**修改密码**：

1. 先解锁密码
2. 点击「修改密码」
3. 输入旧密码和新密码
4. 点击保存

### 7.4 开机自启

**作用**：系统启动时自动运行微连

**设置方法**：

1. 进入「设置」页面
2. 切换「开机自启」开关
3. 需要管理员权限

### 7.5 托盘最小化

**作用**：关闭窗口时最小化到系统托盘，而非退出

**设置方法**：

1. 进入「设置」页面
2. 切换「托盘最小化」开关
3. 启用后，关闭窗口仅隐藏到托盘

## 8. 数据同步

### 8.1 WebDAV 配置

**支持的服务**：

- 坚果云
- 腾讯微云
- 百度网盘（需第三方 WebDAV 工具）
- 自建 WebDAV 服务器

**配置步骤**：

1. 进入「设置」页面
2. 点击「数据同步」选项
3. 填写配置：
   - **服务器 URL**：WebDAV 服务器地址
   - **用户名**：登录用户名
   - **密码**：登录密码
   - **路径**：同步路径（默认 `/`）
4. 点击「测试连接」
5. 连接成功后点击保存

### 8.2 手动同步

**上传到云端**：

1. 进入「设置」页面
2. 点击「数据同步」选项
3. 点击「上传」按钮
4. 等待上传完成

**从云端恢复**：

1. 进入「设置」页面
2. 点击「数据同步」选项
3. 点击「下载」按钮
4. 确认下载
5. 应用会自动重启

### 8.3 自动同步

**开启自动同步**：

1. 进入「设置」页面
2. 点击「数据同步」选项
3. 勾选「自动同步」
4. 设置同步间隔（默认 5 分钟）

### 8.4 本地备份

**创建备份**：

1. 进入「设置」页面
2. 点击「数据备份」选项
3. 点击「创建备份」
4. 备份文件保存在：

   - Windows: `C:\Users\{用户名}\.wechat-link-agent\backups\`
   - macOS: `/Users/{用户名}/.wechat-link-agent/backups/`
   - Linux: `/home/{用户名}/.wechat-link-agent/backups/`

**恢复备份**：

1. 进入「设置」页面
2. 点击「数据备份」选项
3. 选择备份文件
4. 点击「恢复」
5. 确认恢复

## 9. 常见问题

### 9.1 安装相关

**Q: Windows 提示「无法验证发布者」怎么办？**

A: 点击「更多信息」→「仍要运行」即可。

**Q: macOS 提示「无法打开，因为无法验证开发者」怎么办？**

A: 打开「系统设置」→「隐私与安全性」→「仍要打开」。

**Q: Linux 提示权限不足怎么办？**

A: 添加执行权限：`chmod +x 微连-0.1.0.AppImage`

### 9.2 Agent 相关

**Q: 扫描后没有发现 Agent？**

A: 请确保：
1. Agent 已正确安装
2. Agent 路径在 PATH 环境变量中
3. 点击「重新扫描」按钮

**Q: 切换 Agent 后发送消息没反应？**

A: 请检查：
1. Agent 是否已登录（如 `claude auth login`）
2. Agent 是否配置了 API Key
3. 查看日志文件排查问题

**Q: Agent 回复很慢？**

A: 可能原因：
1. 网络连接不稳定
2. Agent 服务器繁忙
3. 模型参数设置过高

### 9.3 微信相关

**Q: 扫码登录失败？**

A: 请检查：
1. 网络连接正常
2. 微信版本较新
3. 重新生成二维码并重试

**Q: 微信中发送消息没回复？**

A: 请检查：
1. 微信连接状态是否为「已连接」
2. 当前是否已选择 Agent
3. Agent 是否在线

**Q: 如何在微信中查看当前 Agent？**

A: 发送消息：`/help`

### 9.4 数据相关

**Q: 数据库文件在哪里？**

A: 默认位置：
- Windows: `C:\Users\{用户名}\.wechat-link-agent\wla.db`
- macOS: `/Users/{用户名}/.wechat-link-agent/wla.db`
- Linux: `/home/{用户名}/.wechat-link-agent/wla.db`

**Q: 如何迁移数据到新电脑？**

A: 使用 WebDAV 同步或手动复制数据库文件：
1. 关闭微连
2. 复制数据库文件到新电脑
3. 启动微连

**Q: 忘记密码怎么办？**

A: 密码重置会清除所有 API Key，需重新配置：
1. 关闭微连
2. 删除数据库文件
3. 重新启动微连

### 9.5 性能相关

**Q: 应用占用内存过高？**

A: 可能原因：
1. 会话消息过多（尝试删除旧会话）
2. Agent 子进程未正常退出（重启应用）

**Q: 启动速度很慢？**

A: 可能原因：
1. 磁盘 I/O 较慢
2. 数据库文件过大（尝试清理旧数据）
