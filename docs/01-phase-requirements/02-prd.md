## 1. 产品概述

### 1.1 核心功能

微连是一款 Electron 桌面应用，核心功能包括：

1. **Agent 管理**：扫描、安装、切换本地 AI Agent CLI 工具
2. **微信集成**：扫码登录微信，通过微信发送指令调用 Agent
3. **会话管理**：多会话支持、消息历史、导出功能
4. **配置中心**：主题、语言、工作目录、WebDAV 同步等
5. **工具箱**：LLM 供应商管理、API Key 加密存储
6. **商城**：展示合作 AI 工具，提供安装链接

### 1.2 技术架构

- **前端框架**：React 18 + TypeScript
- **桌面框架**：Electron 33
- **状态管理**：Zustand
- **UI 组件**：Radix UI + Tailwind CSS
- **数据库**：SQLite (sql.js)
- **国际化**：i18next

## 3. 业务流程图

### 3.1 应用启动流程

```
应用启动
  ├─> 数据库初始化 (schema.sql + seed.sql)
  ├─> 扫描 PATH 发现 Agent
  ├─> Agent Manager 初始化
  ├─> 尝试恢复微信连接
  ├─> 尝试恢复 WebDAV 配置
  └─> 创建主窗口 + 托盘
```

### 3.2 微信消息处理流程

```
微信收到消息
  ├─> 命令路由 (/agent, /clear, /help)
  │   └─> 执行对应操作并回复
  └─> 普通消息
      ├─> 获取当前 Agent
      ├─> 创建/切换会话
      ├─> 调用 Agent.send()
      ├─> 流式输出推送
      └─> 保存消息到数据库
```

### 3.3 Agent 切换流程

```
用户选择 Agent
  ├─> 更新 AgentManager 当前 Agent
  ├─> 通知渲染进程更新状态
  ├─> 加载该 Agent 的会话列表
  └─> 创建新会话或切换到最新会话
```

## 5. 页面交互逻辑

### 5.1 三栏布局

```
┌────────┬─────────────────────┬──────────────────────────────┐
│        │   Agent List /      │                              │
│  Nav   │   Session List      │      Chat / Settings         │
│  Bar   │   / Store /         │      / Agent Manager         │
│ 60px   │   Settings List     │                              │
│        │ 200-500px (可拖拽)  │         flex: 1               │
└────────┴─────────────────────┴──────────────────────────────┘
```

### 5.2 核心交互

- **导航切换**：点击导航栏图标，第二栏切换显示对应列表
- **选中项切换**：点击列表项，第三栏切换显示详情页
- **消息发送**：
  - 桌面端：输入框 + 回车发送
  - 微信端：直接发送消息到机器人
- **流式响应**：
  - Agent 输出通过 IPC 推送到渲染进程
  - 渲染进程逐字显示（Markdown 渲染）

## 6. 异常边界条件

### 6.1 Agent 相关

| 场景 | 处理方式 |
|------|----------|
| Agent CLI 未安装 | 显示安装提示，提供一键安装脚本 |
| Agent 调用失败 | 显示错误信息，记录日志 |
| Agent 响应超时 | 超过 5 分钟中断，提示重试 |
| Agent 退出异常 | 重新启动进程，恢复会话 |

### 6.2 微信相关

| 场景 | 处理方式 |
|------|----------|
| 微信连接断开 | 自动重连（指数退避） |
| 二维码过期 | 重新生成二维码 |
| 消息发送失败 | 重试 3 次，失败后提示用户 |
| 多媒体上传失败 | 提示文件过大或格式不支持 |

### 6.3 数据相关

| 场景 | 处理方式 |
|------|----------|
| 数据库损坏 | 使用最近备份恢复 |
| 加密密钥丢失 | 提示重置密码，清空 API Key |
| WebDAV 同步失败 | 保留本地数据，提示检查网络 |

## 7. 非功能性需求

### 7.1 性能要求

- 应用启动时间 < 3 秒
- 消息发送响应时间 < 1 秒
- 流式输出延迟 < 500ms
- 支持并发处理 10+ 会话

### 7.2 安全要求

- API Key 本地 AES-GCM 加密
- 主密码保护敏感数据
- 不上传用户数据到云端（除非用户主动同步）
- 日志不包含敏感信息

### 7.3 兼容性要求

- **操作系统**：Windows 10+、macOS 12+、Linux (Ubuntu 20.04+)
- **Node.js**：v18+
- **微信**：支持最新版微信
